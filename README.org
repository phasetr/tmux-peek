* tmux-peek

tmux-peek is a small Emacs Lisp library for inspecting live tmux state
asynchronously.

This is a personal tool for GNU Emacs 30.1 and tmux 3.6a.  It is designed
around the needs of [[https://github.com/phasetr/enkan-repl][enkan-repl]], but remains independent.

** Scope

- Query tmux sessions, windows, and panes.
- Capture pane text, including one-shot tail-like capture.
- Provide async process helpers.
- Provide only =kill-pane= for explicitly selected pane cleanup.

tmux-peek intentionally does not provide wrappers for =send-keys=,
=new-session=, =new-window=, =attach-session=, =kill-session=,
=kill-window=, or =kill-server=.
